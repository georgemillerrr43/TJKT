# TJKT Website

Website portfolio jurusan Teknik Komputer dan Jaringan (TJKT).

## Dibuat Oleh

Joji

## Teknologi

- React
- TypeScript
- Vite
- Tailwind CSS

## Cara Menjalankan
npm install
npm run dev
npm run build

## Struktur Folder
src/
├── components/
├── pages/
├── project/
├── App.tsx
└── main.tsx

## Routing
/ - Beranda
/programs - Program Keahlian
/projects - Daftar Proyek
/projects/cisco-project - Detail Cisco
/projects/mikrotik-project - Detail MikroTik
/projects/web-project - Detail Web
/projects/iot-project - Detail IoT
/projects/server-project - Detail Server
/projects/fiber-project - Detail Fiber
/gallery - Galeri
/contact - Kontak

## Fitur
Halaman program keahlian dengan perbandingan
Daftar proyek teknis siswa
Detail proyek per halaman
Animasi scroll reveal
Desain responsif

## Catatan
Proyek ini dibuat untuk keperluan pembelajaran TJKT. dan sebagai sarana agar siswa TJKT SMK MITRA KARYA menjadi lebih bersemangat untuk menjalankan pembelajaran dan eksplorasi dunia digital atau teknologi
